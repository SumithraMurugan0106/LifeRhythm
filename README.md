# 🌿 LifeRhythm – AI-Powered Productivity & Digital Wellness Assistant

LifeRhythm is an AI-powered mobile application developed using **Flutter**, **Firebase**, and **Google Gemini AI** to help users build healthy digital habits, improve productivity, and achieve their career goals. The application analyzes daily activities such as screen time, productivity, sleep, and wellness to generate personalized insights and recommendations.

---

## 🚀 Features

- 🔐 Secure User Authentication (Firebase Authentication)
- 👤 User Onboarding & Profile Management
- 📊 Digital Wellness Analytics Dashboard
- 📱 Screen Time & App Usage Monitoring
- 💤 Sleep & Productivity Tracking
- 🎯 Career Goal Planning
- 🤖 AI Coach powered by Google Gemini AI
- 📝 Personalized Daily Planner
- ⏱️ Pomodoro Focus Timer
- 🏆 Achievement & Progress Tracking
- 📈 Life Score Calculation
- ☁️ Cloud Data Storage using Firebase Firestore

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/LifeRhythm.git
```

### Navigate to project

```bash
cd LifeRhythm
```

### Install packages

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```

---

## 🔑 Environment Setup

Create the following file:

```
lib/core/config/api_keys.dart
```

```dart
class ApiKeys {
  static const String geminiApiKey = "YOUR_GEMINI_API_KEY";
}
```

Configure Firebase by adding:

- `android/app/google-services.json`
- `ios/Runner/GoogleService-Info.plist`
